# Pokémon Showdown Clone

- browser-based Pokemon battle game clone inspired by "Pokémon Showdown".
- built using *HTML, CSS, and JavaScript, and integrates data from the **PokéAPI*.

## Features

* Fetch Pokémon data using a public API
* Display Pokémon with basic stats
* Filtering, sorting, and searching using array methods to get moves/attacks/stats etc
* Clean and responsive UI using CSS

## Tech Stack

* HTML
* CSS
* JavaScript (Vanilla)
* PokéAPI (https://pokeapi.co/)

## Future Improvements

* Add Pokémon moves and abilities
* Implement Pokémon types and type effectiveness
* Build a basic battle system
* Improve UI/UX to more closely match Pokémon Showdown
* Add team building functionality
